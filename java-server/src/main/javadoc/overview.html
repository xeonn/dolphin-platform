This is the documentation of the public server API of Dolphin Platform.

The Dolphin Platform is a framework that implements the presentation model pattern and provides a modern way to create enterprise applications. The Platform provides several client implementations that all can be used in combination with a general sever API. By doing so you can create enterprise application with a single server and several desktop, web and mobile client implementations. Here the Dolphin Platforms define a mechanism to automatically snchronize models between the server and the client.

This module contains the public and general API of the Dolphin Platform server API. To use the Dolphin Plaftorm in your server you need a container specific implementation. Currently the Dolphin Platform provides implementations for Spring and JavaEE. Since you need to add this specific modules to your classpath you will never need to code against them. All the classes and interfaces that are needed to create a server application or module that is based on the Dolphin Platform can be found in the public API that is documented in here.

The perfect start of this API documentation is the DolphinController annotation that is used to define server side controllers.

You can find the documentation of the different modules here.

Additional information can be found on the Dolphin Platform website.
